## Ephemeral Nodes

These nodes appear for 4 hours. They can be reappear in that 4 hours, if you touch 3 different nodes surrounding them.

* *Sea of Clouds (2000)* 
* *Coerthas(0000)*
* *Hinterlands(0400)* 
* *Forelands(0800)*
* *??? (1200)*
* *Churning Mists (1600)*
